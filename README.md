**Feedbacks** is an easy way to show UI/UX feedback.

```
$ pod try Feedbacks
```

## Requirements

- iOS 10.0+
- Xcode 9.3+
- Swift 4.2

## Installation

### CocoaPods
To install with [CocoaPods](http://cocoapods.org/), simply add this in your `Podfile`:
```ruby
use_frameworks!
pod "Feedbacks"
```
